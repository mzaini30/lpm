# lpm

Lua version `npm`.

## Installation

```bash
luarocks install lpm
```

## Using

### Init

```bash
lpm init
```

### Install from Rockspec

```bash
lpm i
```

### Install New Packages

```bash
lpm i package1 package2 package3
```

### Clean

```bash
lpm c
```
